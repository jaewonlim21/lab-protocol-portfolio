# SDS-PAGE

## Overview
Gel electrophoresis workflow used before or after purification to assess protein band pattern, size range, and fraction quality.

## Purpose
To compare binding, wash, elution, and concentrated fractions and judge purification success.

## Standard structure
1. Prepare samples and loading buffer.
2. Denature under defined temperature conditions.
3. Set up MES running buffer and gel apparatus.
4. Load ladder and samples.
5. Run gel.
6. Wash, stain, and image.

## Critical parameters
- calculate loading mix carefully
- distinguish pellet and supernatant denaturation conditions if needed
- maintain consistent sample naming
- document lane assignment before loading

## Troubleshooting
### Smearing
Review sample concentration, salt carryover, and denaturation conditions.

### Weak or absent band
Check loaded mass, sample handling, and whether the protein is expected in that fraction.

### Inconsistent lanes
Review pipetting consistency and whether viscous loading buffer was mixed fully.
