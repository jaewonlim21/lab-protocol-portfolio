# Recombinant Lectin Workflow

## End-to-end flow

```mermaid
flowchart LR
  A[Single colony or frozen stock] --> B[BMGY seed culture]
  B --> C[BMMY induction]
  C --> D[Methanol feeding and OD tracking]
  D --> E[Harvest]
  E --> F[Supernatant or pellet processing]
  F --> G[Ni-NTA purification]
  G --> H[SDS-PAGE]
  H --> I[Western blot / quantification]
```

## Why include workflow pages
A workflow page helps employers see that the methods are connected, not isolated.
